# Core Python Practice Questions

## Level 1: Basics (Loops, Conditions)

1. Print numbers from 1 to N
2. Print all even numbers between 1 and N
3. Find sum of first N natural numbers
4. Check if a number is even or odd
5. Print multiplication table of a number
6. Reverse a number (e.g., 123 → 321)
7. Count digits in a number

Goal: Strong control over loops and conditions.

## Level 2: Strings

1. Check if a string is a palindrome
2. Count vowels and consonants in a string
3. Find frequency of each character in a string
4. Remove duplicate characters from a string
5. Find the longest word in a sentence
6. Convert a string to title case (first letter capital)
7. Count number of words in a sentence

Goal: Build logic using string manipulation.

## Level 3: Lists

1. Find largest and smallest element in a list
2. Remove duplicates from a list
3. Find second largest number
4. Reverse a list
5. Merge two lists
6. Find common elements between two lists
7. Count frequency of elements in a list

Goal: This directly prepares you for working with tabular data later.

## Level 4: Dictionaries

1. Count frequency of elements in a list using a dictionary
2. Merge two dictionaries
3. Find key with maximum value
4. Invert a dictionary (swap key and value)
5. Group elements based on category (e.g., students by grade)

Goal: This builds thinking similar to grouping and aggregation in pandas.

## Level 5: Functions and Logic

1. Write a function to calculate factorial
2. Generate Fibonacci series using a function
3. Check if a number is prime
4. Return all prime numbers in a given range
5. Write a function to flatten a nested list

Goal: Write reusable and clean code.

## Level 6: Practical Problems (Important)

1. Given a list of sales values, calculate total revenue
2. Find the most frequent item in a list
3. Given student marks, calculate average and assign grades
4. Filter out invalid values (negative numbers, missing values)
5. Count occurrences of each word in a paragraph

Goal: Start thinking like someone working with real data.

# How to Practice

- Attempt each problem yourself for at least 15–20 minutes
- If stuck, look at the solution
- Then rewrite the solution without looking
- Revisit the same problem after 2–3 days

# Daily Practice Plan

- Solve 5–6 questions per day
- Mix different levels (not just easy ones)
- Revise older problems regularly

# When You’re Ready to Move Ahead

Before moving to libraries like pandas, you should be able to:

- Solve list and dictionary problems without help
- Write functions comfortably
- Handle basic edge cases
