# Pandas Practice Questions (Core to Advanced)

## 1. Basic DataFrame Understanding

1. Load a CSV file into a DataFrame
2. Display first 5 rows
3. Display last 5 rows
4. Check shape of dataset
5. Get column names
6. Check data types of each column
7. Get summary statistics of numeric columns

## 2. Data Selection & Filtering

1. Select a single column from DataFrame
2. Select multiple columns
3. Filter rows where a column value > X
4. Filter rows based on multiple conditions (AND / OR)
5. Get all rows where value is null
6. Get all rows where value is NOT null
7. Filter rows based on string matching (e.g. city = "Delhi")

## 3. Data Cleaning

1. Find number of missing values in each column
2. Fill missing values with mean
3. Fill missing values with median
4. Drop rows with missing values
5. Rename column names
6. Convert data types (e.g. string → int)
7. Remove duplicate rows

## 4. GroupBy (Very Important for Interviews)

1. Group data by one column and find sum
2. Group data by one column and find mean
3. Group by multiple columns
4. Count number of records in each group
5. Find max/min value in each group
6. Find top performing group

## 5. Sorting & Ranking

1. Sort DataFrame by one column ascending
2. Sort DataFrame by multiple columns
3. Sort values in descending order
4. Find top 5 rows based on a column
5. Create ranking based on values

## 6. New Columns (Feature Engineering Basics)

1. Create a new column using existing columns
2. Apply condition to create a new column
3. Categorize values (e.g. High/Medium/Low)
4. Calculate percentage from existing columns
5. Apply lambda function to a column

## 7. Aggregation (Very Important)

1. Find sum, mean, max, min of a column
2. Apply multiple aggregations at once
3. Aggregate after grouping
4. Compare groups using aggregation

## 8. Real-World Thinking Questions

1. Find highest selling product
2. Find city with highest revenue
3. Find customer who spent the most
4. Find average sales per category
5. Identify underperforming products
6. Find percentage contribution of each group

## 9. Advanced Filtering

1. Filter top N records per group
2. Find rows where values fall within a range
3. Filter using multiple conditions across columns
4. Extract partial string matches
5. Filter based on datetime column (if available)

## 10. Mini Challenge Questions

1. Create a DataFrame from scratch and analyze it
2. Simulate missing data and clean it
3. Create a ranking system for customers
4. Find trends using grouping
5. Build a small “report” from dataset

# How to Practice This Properly

Do NOT just read these.

Follow this method:

1. Try solving without help (15–20 min)
2. If stuck, check hint or solution
3. Rewrite solution without looking
4. Repeat after 2 days

# Minimum Target Before Moving On

You should be able to:

- Clean any dataset
- Use groupby confidently
- Filter data without confusion
- Create new columns easily

If you can’t do this, moving to SQL or ML will feel difficult.
