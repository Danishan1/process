# Advanced Pandas Practice Questions

## 1. Complex Filtering & Conditions

1. Filter rows where a column is above its overall mean
2. Find rows where multiple numeric conditions are true across columns
3. Filter top 10% of values in a column
4. Select rows based on string patterns (contains, startswith, endswith)
5. Filter rows where a value appears more than N times in dataset

## 2. GroupBy (Advanced Level)

1. Find top 2 records per group
2. Compute multiple aggregations (sum, mean, max) per group
3. Find group-wise percentage contribution to total
4. Identify groups with increasing/decreasing trends
5. Rank values within each group

## 3. Window-like Operations (Very Important)

1. Calculate cumulative sum of a column
2. Calculate rolling average (window = 3 or 5)
3. Find difference between consecutive rows
4. Calculate percentage change over time
5. Identify spikes in data using rolling statistics

## 4. Multi-Column Logic

1. Create a column based on multiple conditions
2. Compare two columns row-wise and create flags
3. Compute interaction features (e.g. A \* B / C)
4. Normalize a column using another column
5. Conditional replacement across multiple columns

## 5. Missing Data (Advanced Handling)

1. Fill missing values using group-wise mean
2. Forward fill and backward fill in sequences
3. Interpolate missing numeric values
4. Detect patterns in missing data
5. Replace missing values based on condition

## 6. Joins and Merging (Very Important in Jobs)

1. Inner join two datasets
2. Left join and analyze missing matches
3. Right join use-case scenario
4. Full outer join and cleanup
5. Merge on multiple keys

## 7. Reshaping Data

1. Pivot table creation
2. Melt (wide to long format)
3. Stack and unstack operations
4. Convert aggregated data back to raw form
5. Reshape dataset for reporting format

## 8. Real-World Business Questions

1. Find revenue contribution by each category and rank them
2. Identify customers who stopped purchasing (churn-like logic)
3. Find most consistent performer across months
4. Detect outliers in sales data
5. Find best performing region per product

## 9. Time-Based Analysis (If datetime exists)

1. Convert column to datetime format
2. Extract month, year, day
3. Group data by month and analyze trends
4. Find weekly or monthly growth rate
5. Compare performance across time periods

## 10. Interview-Level Problem Solving

1. Find second highest value per group
2. Detect duplicates based on multiple columns
3. Find rows that violate business rules
4. Create a scoring system based on multiple features
5. Build a summary report from raw dataset

# How to Practice Advanced Pandas

Do this strictly:

- Pick 2–3 questions daily
- Solve without looking at syntax
- If stuck, break problem into steps
- Rebuild solution from scratch after learning

# What “Good” Looks Like Here

You should be able to:

- Think in transformations, not loops
- Combine groupby + filtering + aggregation
- Explain _why_ each step is used
- Build small analytical reports from raw data

# Important Reality Check

If you can do these comfortably, you are already at:

- Junior data analyst level
- Interview-ready for pandas rounds

If not, stay here longer before moving to machine learning.
