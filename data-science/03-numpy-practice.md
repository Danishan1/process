# NumPy Practice Plan (Practical + Interview-Oriented)

## Phase 1: Basics (Very Important)

Start with understanding arrays.

### Practice:

1. Create 1D and 2D arrays
2. Check shape, size, dimensions
3. Reshape an array
4. Create arrays of zeros, ones, identity matrix
5. Generate random numbers

### Goal:

- Be comfortable creating and inspecting arrays

## Phase 2: Indexing & Slicing (Core Skill)

This is heavily used later in data work.

### Practice:

1. Access specific elements in 2D array
2. Slice rows and columns
3. Reverse an array
4. Extract sub-matrix from a matrix
5. Boolean indexing (filter values > X)

### Goal:

- Retrieve and filter data efficiently

## Phase 3: Operations (Where NumPy shines)

### Practice:

1. Element-wise addition, subtraction, multiplication
2. Matrix multiplication
3. Find mean, median, standard deviation
4. Sum across rows and columns
5. Normalize an array

### Goal:

- Replace loops with vectorized operations

## Phase 4: Real Data-Type Problems

Now connect to practical thinking.

### Practice:

1. Given an array of marks, find:
   - average
   - highest and lowest

2. Replace all negative values with zero
3. Find indices where values > threshold
4. Standardize data (mean = 0, std = 1)
5. Combine two datasets (arrays)

## Phase 5: Interview-Level Problems

1. Find duplicates in an array
2. Sort array without using built-in sort (logic + NumPy combo)
3. Find top N values
4. Count frequency of elements
5. Rotate an array

# How to Practice NumPy Properly

- Do not use loops unless necessary
- Always ask: “Can this be vectorized?”
- Practice writing shorter, efficient code

# Daily Plan (2–3 Days Enough)

Day 1:

- Basics + array creation
- Indexing

Day 2:

- Operations
- Filtering

Day 3:

- Real problems + mini practice

# Common Mistakes to Avoid

- Treating NumPy like lists
- Overusing loops
- Ignoring shape mismatches
- Not understanding broadcasting

# When You’re Done

You should be able to:

- Work with arrays confidently
- Perform operations without loops
- Prepare data for pandas
